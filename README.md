# Mortgage Calculator

## How to run

```shell

npm install

npm run build

```

Builds files from `src/` (JS, SCSS) to `dist/`


To see the application, access `dist/index.html` from a browser.

## Live (development) mode 

```shell
npm run dev
```

Watches for changes inside `src/` files

## Run tests

```shell
npm test
```
